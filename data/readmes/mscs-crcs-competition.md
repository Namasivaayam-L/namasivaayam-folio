# MSCS CRCs Competition Platform

## Overview
A full-stack web application for the MSCS CRCs competition, featuring data visualization, society registration, forms management, and interactive maps. Built with React/TypeScript frontend and Node.js/Express backend with MongoDB database.

## Features
- Data visualization and charting capabilities
- Society registration and management system
- Form management and PDF viewing
- Interactive maps with Google Maps API integration
- User authentication and authorization
- Admin dashboard for competition management

## Tech Stack
- React (TypeScript)
- Node.js
- Express
- MongoDB
- Material UI
- Chart.js
- Google Maps API
- PDF.js
- JWT authentication
- Bcrypt password hashing

## Architecture
The platform consists of:
- Client-side: React/TypeScript frontend with Material UI components
- Server-side: Node.js/Express backend with MongoDB database
- Authentication: JWT-based user authentication
- Data Visualization: Chart.js for interactive charts and graphs
- Maps: Google Maps API integration for location-based features

## Setup
```bash
# Client setup
cd client
npm install
npm start

# Server setup
cd server
npm install
npm start
```
